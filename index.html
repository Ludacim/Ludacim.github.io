Web hosting

Un web hosting es un servicio que permite a los usuarios publicar su sitio web en Internet. En otras palabras, es un espacio en línea donde se almacena el contenido de un sitio web y se hace accesible a los visitantes de la web. Los proveedores de alojamiento web ofrecen diferentes planes de alojamiento que varían en precio y características.
GitHub es una plataforma de desarrollo colaborativo de software que utiliza el sistema de control de versiones Git. Es un portal para gestionar las aplicaciones que utilizan el sistema Git. GitHub ofrece cuentas personales para individuos y organizaciones para equipos de personas que trabajan juntos. Los usuarios pueden alojar y revisar el código, administrar proyectos y crear software junto con otros desarrolladores.
